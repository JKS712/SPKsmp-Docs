<details>
    <summary>使用方式</summary>
        * 法一: 直接投入物品
          * 在乖乖這裡對著石牆投入物品\
            ![alt text](image-45.png)
        * 法二: 使用假人
          1. 轉角處音符盒其中一個為假人投入鈕，按下後石牆將會縮回\
            ![alt text](image-46.png)
          2. 將假人召喚在此地，右鍵開啟其背包置入物品\
            ![alt text](image-47.png)
          3. 再按一次音符盒即可將假人推入
          * 玩家若不慎掉入，可按住`Shift`從通道離開
</details>
