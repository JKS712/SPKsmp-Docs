# 伺服器端模組列表
本篇列出伺服器已安裝的模組供玩家了解
:::warning 這並不是玩家需要裝的模組
玩家所使用的模組請查看 [整合包模組一覽](/docs/04.mod/03.use/02.client-mod.md)
:::
* 點擊模組名稱可跳轉至MC百科查看中文介紹
* 若MC百科未收錄則會放上官方網頁連結
