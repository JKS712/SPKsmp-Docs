## 永晝機
> 主世界 (760, Y, -400) (主城內)\
> ![alt text](image-19.png)
<details>
    <summary>使用方式</summary>
      1. 呼叫假人
              ```
              /player Sleep spawn at 764.31 67.88 -411.53 facing 2070.90 7.20 in minecraft:overworld
              ```
      2. 讓假人持續右鍵
              ```
              /player Sleep use continuous
              ```
      * 若要關閉永晝機
              ```
              /player Sleep stop
              ```
</details>
:::success 不用讓此假人下線 主要是用來加載主城的
:::
